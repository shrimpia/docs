---
title: おすわり
---

本サーバーでは、階段、ハーフブロック、カーペットなどに対し、何もアイテムを持たない状態で右クリック（タッチデバイスではタップ）することで、そのブロックに座ることができます。

また、 `/sit` コマンドで、どこでも座れます！

本プラグインでは、他にも様々なポーズを取れるコマンドがあります。

| コマンド | 説明 |
| --- | --- |
| `/sit` | おすわり |
| `/crawl` | 匍匐前進（Bedrock版では不安定） |
| `/lay` | 仰向けになる（Bedrock版非対応） |
| `/spin` | その場で高速回転 |
