## ワールド間移動
- `/spawn` : 5秒待った後、スポーン地点に移動します。
- `/goto main` : 5秒待った後、メインワールドに移動します。
- `/goto sub` : サブワールドに移動します。
- `/goto bed` : 5秒待った後、最後に使用したベッドか、リスポーンアンカーのある場所に移動します。
